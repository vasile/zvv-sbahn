Zürich S-Bahn Network Expansion

Tweet

This application shows the Zürich S-Bahn Network evolution from 1990 to present day. The data is gathered from ZVV and Wikipedia datasources.
Created by Vasile Coțovanu, the project it's powered by GeoAdmin API, Boostrap 3.0 and open-sourced on Github.
